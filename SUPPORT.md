# Support Guide

## Where to Ask

For usage help, open a GitHub issue using the `Question / Help` template.

For bugs, use the `Bug report` template.
For feature ideas, use the `Feature request` template.

## What to Include

Please include:

- your goal
- exact commands you ran
- full error output (with secrets removed)
- OS and Node version
- `droxy version` output

Useful diagnostics:

```bash
node droxy.js --help
node droxy.js status --json
```

## Security Issues

Do not report vulnerabilities in public issues.

Use:

- `https://github.com/mihneaptu/droxy-cli/security/advisories/new`
